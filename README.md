# 🍫 Receita de Bolo de Chocolate

Projeto simples desenvolvido com HTML para apresentar uma receita de bolo de chocolate de forma organizada e fácil de visualizar.

---

## 📖 Sobre o Projeto

Esta página exibe uma receita de bolo de chocolate contendo:

- Imagem do bolo
- Lista de ingredientes
- Ingredientes da cobertura
- Lista de utensílios
- Modo de preparo

O objetivo do projeto é praticar a estrutura básica do HTML utilizando títulos, imagens e listas.

---

## 🛠️ Tecnologias Utilizadas

- HTML5

---

## 📂 Estrutura da Página

- Título da receita
- Imagem ilustrativa
- Ingredientes
- Cobertura
- Utensílios
- Modo de preparo

---

## 🎯 Objetivo

Praticar conceitos básicos de HTML como:

- Títulos (`h1`, `h2`)
- Imagens (`img`)
- Listas (`ul` e `li`)
- Organização de conteúdo

---

## ▶️ Como Executar

1. Baixe os arquivos do projeto.
2. Abra o arquivo `index.html` no navegador.
3. Visualize a receita.
